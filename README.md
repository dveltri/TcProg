# TcProg
Software to program Traffic Controller made in ElectronJs

## Linux
> Lo primero que hay que hacer es:
- npm install
> Luego de instaladas todas la dependencias:
- npm start
> Y para generar el paquete instalador de wibndows:
- mkWin32.sh
